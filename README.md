# College Management API
Simple FastAPI REST API with JWT auth and Docker.

## Features
- JWT Authentication
- Student CRUD operations
- Docker support
- HTML test interface

## Quick Start
```In bash
docker-compose up --build
```

Access:
- Test UI: Open `index.html` 
- API Docs: http://localhost:8000/docs
